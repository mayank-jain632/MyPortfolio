# Mayank Jain — React Portfolio

A polished, dark‑mode portfolio for Mayank Jain, built with **React + Vite + TailwindCSS** and pre‑wired for motion with **React Bits**.

## Getting started

```bash
npm install
npm run dev
```

Then open the printed localhost URL in your browser.

## Tech stack

- React + Vite
- TailwindCSS
- React Bits (`@appletosolutions/reactbits`) — installed and ready to use

You can customize content under `src/data` and tweak layout/sections in `src/components`.
